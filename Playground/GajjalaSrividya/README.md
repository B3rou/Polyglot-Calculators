This is a simple Command-Line Calculator written in Java that performs basic mathematical operations like addition, subtraction, multiplication, division, and modulo.
How to run:
1.Compile the program : javac GajjalaSrividya\SimpleCalculatorCLI.java
2.Run the program: java GajjalaSrividya\SimpleCalculatorCLI
### Example usage:
Enter integer 1: 10
Enter integer 2: 5
Enter operation (+, -, *, /, %): /
Result: 2
Do you want to perform another operation? (y/n): n
Calculator closed.
